## 🚀 GreenStack — Modern Umbraco Hosting

Our next-generation hosting platform, **GreenStack**, launched in January 2026 and brings a fully containerised, cloud-native stack designed specifically for Umbraco workloads.

| Tier | Best for |
|------|----------|
| **Core** | Small sites & personal projects |
| **Pro** | Growing agencies & production sites |
| **Scale** | High-traffic & enterprise deployments |

🔗 [umbhost.net — Modern Umbraco Hosting](https://umbhost.net/gb/sustainable-cloud-hosting/modern-umbraco-hosting)

### What's under the hood

- 🐳 **Docker Swarm** orchestration via Portainer EE
- 🗄️ **GlusterFS** replicated storage for high availability
- 🌐 **Cloudflare Tunnels** & Traefik reverse proxy
- 📊 **Prometheus + Grafana** monitoring
- 🔐 **Automated TLS**, custom hostnames, and edge security
- 🗃️ **Azure SQL** backup/export (BACPAC) for database peace of mind

---

## 🛠️ UmbPanel

**UmbPanel** is our in-house hosting management platform powering the GreenStack experience — giving customers a clean, developer-friendly interface to manage their Umbraco environments.

**Key features:**

- Real-time container stats & resource monitoring
- Web-based terminal (xterm.js)
- Docker Registry management
- Custom hostname management with Cloudflare integration
- Mailgun/SMTP email configuration
- Service Access Invitations for team collaboration
- Azure SQL database backup & export
- Environment variable management
